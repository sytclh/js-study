1. async 只对外部文件有效
  2. charset 表示通过src属性指定的代码的字符集。几乎不用
  3. defer 表示脚本可以延迟到文档完全被解析和显示之后执行。只对外部文件有效。
  4. language 原用于表示编写代码使用的脚本语言（JavaScript或VBScript），已废弃
  5. src 表示包含要执行代码的外部文件
  6. type 可以看成language的替代属性；表示编写代码使用的脚本语言的内容类型（也称为MIME类型）默认值是text/javascript

不要在嵌入的JavaScript代码中的任何地方出现"</script>"，也不要将script标签写成<script xxx="xxx" />

不要将script标签放在head内部，这样呈现页面会出现延迟，需要等待下载解析完成。所以尽量把script标签放在body内容的底部
