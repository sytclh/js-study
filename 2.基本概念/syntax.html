1. 区分大小写
  2. 标识符：指变量、函数、属性的名字，或者函数的参数。推荐使用驼峰大小写格式。不能把关键字、保留字、true、false、null用作标识符
    • 第一个字符必须是字母、下划线或美元符号
    • 其它字符可以是字母、下划线、美元符号或数字
  3. 注释：
    // 单行注释
    /*
    * 多行(块级)注释以(/*)开头，以(*/)结尾。其它(*)都不是必须的，提高可读性
    */
  4. 严格模式："use strict";这是一个编译指示，用于告诉JavaScript引擎切换到严格模式。写在函数内部的上方
  5. 语句：以分号结尾，如果省略则由解析器确定语句的结尾。多条语句的代码块以左花括号({)开头，以右花括号(})结尾
